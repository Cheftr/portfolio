About the Project

This is a self-guided project to create a strategic dashboard for a CPO of a fictional company. The request accompanying this dataset is as follows:

We would like to limit employee attrition to a maximum of 10% due to retirement and 25% due to termination as these frequently open our company up to additional pay-outs to employees. These company-wide goals may be useful to you in building key aspects of the dashboard.

In this project I used functions such as FILTER() and SORT() to manipulate the dataset given based on triggers of the selections on the dashboard. Using criteria based functions like AVERAGEIF() and COUNTIF() I aggregate over these filtered ranges to create dynamic visuals for the dashboard.
