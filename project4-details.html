About the Project

This project focused on using the Iris dataset in Power BI to explore and visualize key data points. The dataset contains measurements of various iris flower species, such as sepal and petal dimensions, making it ideal for data classification and analysis. Using Power BI, I created insightful visuals that highlight trends and differences within the data. The platform’s tools allowed me to efficiently manipulate and model the data, showcasing Power BI’s powerful capabilities for delivering clear and meaningful insights through data visualization.
